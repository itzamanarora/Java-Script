Fetch APIs

The fetch APIs provides an interface for fetching(sending/receiving) resources.

It uses Request and Response object.

The fetch() method is used to fetch a resources(data).

let promise = fetch(url, [options])

Understanding Terms

AJAX: is Asynchornous JS & XML(old format)

JSON: is JavaScript Oject Notation(Current Format)

json() method: returns a second promise that resolves with the result of parsing the response body text
