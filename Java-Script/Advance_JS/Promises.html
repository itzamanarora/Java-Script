Promises

Promises: Promises is for 'eventual' completion of task. It is an object in JS.

It is a solution to callback hell.

let promise = new promise((resolve, reject) => { });

NOTE: resolve and reject are callback provided by JS.

A js promise object can be:

* PENDING: The result is undefined.

* RESOLVED: The result is a value(fulfiled)

* REJECTED: The result is an error object.

NOTE: Promise has state(pending, fulfilled), and some result(result for resolve & error for reject).
